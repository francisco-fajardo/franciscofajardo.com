# Inicio

Esta es la página de inicio de franciscofajardo.com

## Compilar

Primero instala las dependencias:

```sh
npm install
```

Luego compila el proyecto en modo producción:

```sh
npm run build-prod
```

Para crear una versión en modo desarrollo ejecuta esto:

```sh
npm run build-dev
```
